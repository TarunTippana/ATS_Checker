# LLM-Based Resume ATS Scorer and Suggestor

You can get the ats score for the resume in the web applications but what about the suggestions and what about the amendments that you need to do.
for that you need to pay right. I Found the solution for this.
Using this model you can get the free suggestions and areas you need to improve etc..,

# How to use
1. Download the file
2. install the dependencies
3. run the code in VScode or jupyter notebook with the command : streamlit run resume_checker.py
